Auto generate resume from markdown using
[markdown-resume-js](https://github.com/c0bra/markdown-resume-js).

## setup

1. `npm install`
1. `brew install Caskroom/cask/wkhtmltopdf` for pdf generation

## generate HTML and PDF versions of resume

```shell
npm run build
```
